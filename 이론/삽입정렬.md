### 삽입정렬

#### 모티브

i번째까지 정렬된 상태에서 i + 1번째를 정리하자.

#### 코드

```java

public static void insertSort(int[] arr) {
   int N = arr.length;
   for(int i = 1; i < N; i++) {
    int v = arr[i];

    // v를 삽입할 장소찾기
    int j = i;
    for(; j > 0; j--) { // 1까지 이동한다.
        if(arr[j - 1] > v) {    // j - 1이 v보다 크다면 j-1 요소를 j번째로 옮긴다.
            arr[j] = arr[j - 1];
        }
        else break;
    }
    arr[j] = v; // j번째에 v를 넣는다.
   }
}
```

### 복잡도

0 + 1 + 2 ... + (N - 1) = 1/2N(N-1)

O(n^2)

경우에 따라선 퀵보다 더 빠르다.

### 장점

in-place sort (swap을 통한 정렬 , 추가 메모리 소모 X)
stable sort (정렬전 같은 값끼리의 요소는 정렬 후에도 위치가 변하지 않는다.)
